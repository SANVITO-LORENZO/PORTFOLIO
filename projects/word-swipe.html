Word Swipe

Daily Puzzle Game in Vanilla JavaScript Daily Puzzle Game in Vanilla JavaScript

Il Progetto The Project

Word Swipe è un mini-gioco web ispirato ai popolari "Daily Puzzle" mobile. L'obiettivo era creare un'esperienza fluida e nativa direttamente nel browser, senza l'uso di framework esterni.

Il gioco è alimentato da un database di migliaia di livelli pre-calcolati ed estrae un puzzle univoco al giorno, salvando i progressi del giocatore per prevenire tentativi multipli. Word Swipe is a web mini-game inspired by popular mobile "Daily Puzzles". The goal was to create a fluid, native-feeling experience directly in the browser, without using external frameworks.

The game is powered by a database of thousands of pre-calculated levels and extracts a unique puzzle per day, saving player progress to prevent multiple attempts.

Tech Stack

  • Vanilla JavaScript (ES6)
  • HTML5 Canvas API
  • Node.js (Algoritmo Generatore)
  • JSON & LocalStorage

Sfide Tecniche Technical Challenges

HTML5 Canvas

Creazione di un motore di rendering in tempo reale per disegnare la "scia" dello swipe in sovrapposizione agli elementi HTML fluidi. Creation of a real-time rendering engine to draw the swipe "trail" overlapping fluid HTML elements.

Generatore in Node.jsNode.js Generator

Sviluppo di uno script locale capace di incrociare 60.000 parole italiane per compilare un database JSON ottimizzato con livelli generati proceduralmente. Development of a local script capable of crossing 60,000 Italian words to compile an optimized JSON database with procedurally generated levels.

Responsive Design

Utilizzo di variabili `vmin` e `clamp()` nel CSS per garantire che la geometria circolare del gioco sia perfetta su ogni dispositivo. Use of `vmin` and `clamp()` CSS variables to ensure the circular geometry of the game is perfect on any device.
